Portfolio Website

A personal portfolio built using HTML and CSS with dark mode toggle

Project Overview

This is a modern, responsive personal portfolio website designed to showcase skills, projects, and education. Built with clean HTML markup and professional CSS styling, it features a sleek dark mode toggle for improved user experience.

Technologies Used

HTML5

Semantic markup for better accessibility

CSS3

Responsive design, CSS variables, and dark mode

JavaScript

Theme toggle and smooth scrolling

Key Features

  • Fully responsive design that works on all devices
  • Dark mode toggle with localStorage persistence
  • Smooth scrolling navigation
  • Semantic HTML for accessibility
  • Professional styling with CSS variables
  • Fast loading and optimized performance

Challenges & Solutions

Challenge: Dark Mode Implementation

Ensuring the dark mode theme persists across page reloads while maintaining contrast ratios.

Solution: Used localStorage to save user preference and CSS variables for easy theme switching.
