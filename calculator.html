Calculator

A functional calculator built using HTML, CSS and JavaScript with basic operations

Project Overview

This is a fully functional web-based calculator that performs basic arithmetic operations. Built with vanilla JavaScript, it demonstrates DOM manipulation, event handling, and user interface design principles. The calculator features a clean, intuitive interface with responsive design.

Technologies Used

HTML5

Calculator layout and button structure

CSS3

Styling and responsive calculator interface

JavaScript

Operation logic and event handling

Key Features

  • Addition, subtraction, multiplication, and division operations
  • Clear/Reset button to start new calculations
  • Display shows current calculation and result
  • Responsive design for mobile and desktop
  • Intuitive user interface with button hover effects
  • Error handling for invalid operations

What I Learned

JavaScript Event Handling

Implemented click event listeners for all buttons to capture user input and perform calculations.

DOM Manipulation

Used JavaScript to dynamically update the display and manage calculator state.

User Experience Design

Created an intuitive interface with clear visual feedback for user interactions.
