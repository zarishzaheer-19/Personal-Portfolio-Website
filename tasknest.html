TaskNest

A responsive web app for organizing homework and project tasks with Local Storage support

Project Overview

TaskNest is a modern task management web application designed specifically for students and project managers. It allows users to create, organize, and manage their homework and project tasks efficiently. Built with responsive design principles, TaskNest works seamlessly on desktop, tablet, and mobile devices. Tasks are saved locally using browser Local Storage, ensuring data persistence.

Technologies Used

HTML5

Semantic markup for task form and lists

CSS3

Responsive design with mobile-first approach

JavaScript

DOM manipulation and Local Storage API

Local Storage

Persistent data storage in browser

Key Features

  • Add, edit, and delete tasks
  • Mark tasks as complete/incomplete
  • Organize tasks by category (Homework, Projects, etc.)
  • Set due dates for tasks
  • Local Storage support for data persistence
  • Fully responsive design for all screen sizes
  • Clean and intuitive user interface
  • Filter tasks by status and category

What I Learned

Local Storage Implementation

Learned how to persist data in the browser using Local Storage API, ensuring tasks are saved even after the page is closed.

Responsive Web Design

Implemented mobile-first design approach with media queries for optimal experience on all devices.

Complex DOM Manipulation

Managed dynamic task creation, editing, and deletion with proper event handling.
